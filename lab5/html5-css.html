HTML5 e CSS

HTML 5

HTML5 (Hypertext Markup Language, versão 5) é uma linguagem de marcação para a World Wide Web e é uma tecnologia chave da Internet, originalmente proposto por Opera Software.É a quinta versão da linguagem HTML. Esta nova versão traz consigo importantes mudanças quanto ao papel do HTML no mundo da Web, através de novas funcionalidades como semântica e acessibilidade.

Elementos semânticos do HTML5 Usou este elemento nesta página? Funcionamento do elemento:
Header done Indica cabeçalho
Nav done Indica menu de navegação
Main done Indica conteúdo principal e único, que não se repete
Article report_problem Indica porção do documento que pode ser vista de forma independente
Section done Especifica uma parte do conteúdo, usada em conjunto com etiquetas
Aside done Indica anotações, publicidade ou outros assuntos desconexos
Footer done Indica rodapé da secção
Details report_problem Define detalhes adicionais que o utilizador pode ver ou esconder, contidos num elemento summary
Meter report_problem Define uma medida escalar estática dentro de um intervalo
Progress report_problem Define uma barra de progresso de uma tarefa
Abbr report_problem Indica o significado de uma abreviatura em conjunto com o atributo title
Address report_problem Informação sobre o contacto do autor

CSS

Cascading Style Sheets (CSS) é um mecanismo para adicionar estilo (cores, fontes, espaçamento, etc.) a um documento web.

Tipo de Seletor Usou o seletor nesta página? Funcionamento do seletor:
Universal done Aplica o estilo a todos os elementos da página
Tipo done Usa o nome do marcador a estilizar
Classe done Associa um nome a um estilo
Id done Permite identificar univocamente um elemento do HTML
Atributo report_problem Aplicam-se a atributos de elementos
Pseudo-Classe done São associados ao estado de um elemento
Pseudo-Elemento done Aplicam-se a partes de elementos

Combinadores de Seletores Usou o combinador nesta página? Funcionamento do combinador:
Seletores descendentes done Todos os p são "dentro" de um div
Seletores filhos done Todos os p que estão "logo dentro" de div
Seletores adjacentes done Todos os p que estão colocados depois de um div
Seletores imediatamente adjacentes report_problem Todos os p que estão colocados "imediatamente" depois de um div
Agrupa seletores done São separados por vírgulas
